# 🎬 MovieFlix - Movie App

A modern and responsive **React Movie App** that allows users to discover, search, and manage their favorite movies.

Built using the **TMDb API**, this app provides real-time movie data, trailers, and a personalized watchlist experience.

---

## 🚀 Live Demo

🔗 https://movie-app-lemon-phi-68.vercel.app/

---

## ✨ Features

* 🔍 **Instant Search** with debounce
* 🎬 **Watch Movie Trailers (YouTube Integration)**
* ❤️ **Add / Remove from Watchlist**
* 💾 **Persistent Watchlist (LocalStorage)**
* 🎭 **Filter Movies by Genre**
* 📄 **Pagination with Smooth UI**
* 🎯 **Hero Banner with Featured Movie**
* ⚡ **Fast & Responsive UI (Mobile Friendly)**
  
---

## 🛠️ Tech Stack

* ⚛️ React (Hooks)
* 🎨 Tailwind CSS
* 🌐 TMDb API
* 📦 Context API (State Management)
* 🎥 YouTube Embed (Trailer)

## ⚙️ Installation & Setup

```bash
# Clone the repo
git clone https://github.com/your-username/movie-app.git

# Navigate into folder
cd movie-app

# Install dependencies
npm install

# Start project
npm run dev
```

---

## 🔑 Environment Setup

Create a `.env` file and add your TMDb API key:

```
VITE_TMDB_API_KEY=your_api_key_here
```
